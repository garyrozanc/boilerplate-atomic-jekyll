Lists

  • Unordered List

    • An unordered list is used when list items do not have any particular order. An unordered list is created with the <ul> tag.
    • <ul> tag also is a two sided tag and closed with </ul> tag.
    • Like an ordered-list, an item for unordered list is listed with the <li> tag.
      • To create a sub list, we will list start with the <ul> tag.
      • Next, we will use the <li> tag to list the first item and </li> to end the first item.
      • Similarly, we will create the rest of the list items with the <li> and </li> tag.
    • Just as you can change the numbering style for an ordered list, you can change the default bullet style for an unordered list.

  • Ordered List

    1. An ordered list is a numbered list. An ordered list can be used whenever a list requires sequence.
    2. Let's assume we want to create an ordered list that displays how Tom likes six colors, the most important color is listed first and the next most important color is listed second, etc.
    3. The <ol> tag starts an ordered list. The <ol> tag is a two-sided tag, which means that it also requires a closing tag </ol>.
    4. The </ol> tag instructs the browser that the ordered list has ended.
    5. Thus each item that needs to be placed in a list, should be placed inside the <li> and </li> tag.
    6. The <ol> tag includes the type attribute that can be used to change numbering style for an ordered list. This attribute can be set to one of the following choices:
    7. <ol type="1"> Arabic numbers (1) - to create list using Arabic numbers. This is the default type for an ordered list.
    8. <ol type="A"> uppercase letters (A) - to create list using uppercase alphabet letters
    9. <ol type="a"> lowercase letters (a) - to create list using lowercase alphabet letters
    10. <ol type="I"> uppercase Roman numerals (I) - to create list using uppercase Roman Numerals
    11. <ol type="i"> lowercase Roman numerals (i) - to create list using lowercase Roman Numerals
